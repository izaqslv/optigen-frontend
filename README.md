# 🧠 OptiGen - Intelligent Fluid Simulation Platform

Aplicação interativa para simulação e análise de sedimentação em fluidos de perfuração utilizando modelos físico–data-driven.

---

## 🚀 Funcionalidades

- Simulação espaço-temporal (c(h,t))
- Comparação de cenários (IA)
- Métricas automáticas (clarificação, estabilidade)
- Indicadores visuais (semáforo)
- Geração de relatório técnico em PDF

---

## 🧪 Tecnologias

- Streamlit
- Python
- Matplotlib / Pandas
